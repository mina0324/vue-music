## 本项目依赖 [Binaryify](https://github.com/Binaryify) 提供的 [NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi.git) 网易云音乐接口

## 项目开始前请先启动接口服务(需要自己搭建)

*****

## 安装依赖
```
npm install
```

### 启动项目
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
